## 2025-05-18 - Vuetify Virtual Table Empty State
**Learning:** `v-data-table-virtual` supports a `no-data` slot that allows for rich empty states, overriding the simple `no-data-text` prop. This is essential for providing actionable guidance in empty lists.
**Action:** Always check for `no-data` slot support in Vuetify data components to upgrade from text-only empty states to illustrative ones.
